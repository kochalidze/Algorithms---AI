рЃЊрЃљрЃљрЃЏрЃбрЃЎрЃўрЃфрЃћрЃЌ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃгрЃўрЃюрЃљрЃЊрЃљрЃЊрЃћрЃЉрЃћрЃЉрЃў
33. A Ріѓ B РЄћ A РѕЕ B = A
рЃЌрЃБ A Ріѓ B, рЃЏрЃљрЃерЃўрЃю рЃЏрЃљрЃерЃўрЃю A-рЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃерЃћрЃЊрЃўрЃА B-рЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃАрЃў, рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ A РѕЕ B рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃўрЃЦрЃюрЃћрЃЉрЃљ A.

34. A Ріѓ B РЄћ A Рѕф B = B
рЃЌрЃБ A рЃерЃћрЃўрЃфрЃљрЃЋрЃА B-рЃерЃў, рЃЏрЃљрЃерЃўрЃю A Рѕф B рЃћрЃарЃЌрЃюрЃљрЃўрЃарЃўрЃљ B-рЃЌрЃљрЃю, рЃарЃљрЃЊрЃњрЃљрЃю A-рЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃДрЃЋрЃћрЃџрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃБрЃЎрЃЋрЃћ B-рЃерЃўрЃљ.

35. B Ріѓ A РЄњ (A \ B) Рѕф B = A
рЃЌрЃБ B рЃљрЃарЃўрЃА A-рЃА рЃЦрЃЋрЃћрЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ, рЃЏрЃљрЃерЃўрЃю A-рЃАрЃњрЃљрЃю B-рЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃўрЃЌ рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃў (A \ B) рЃЊрЃљ B-рЃўрЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃўрЃЌ рЃЋрЃўрЃдрЃћрЃЉрЃЌ A-рЃА.

36. A РѕЕ B = РѕЁ РЄњ (A Рѕф B) \ B = A
рЃЌрЃБ A рЃЊрЃљ B рЃљрЃа рЃљрЃЦрЃЋрЃЌ рЃАрЃљрЃћрЃарЃЌрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў, рЃЏрЃљрЃерЃўрЃю A Рѕф B-рЃЊрЃљрЃю B-рЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ рЃбрЃЮрЃџрЃўрЃљ A-рЃА.

37. A Ріѓ B РЄњ A \ C Ріѓ B \ C
рЃЌрЃБ A рЃљрЃарЃўрЃА B-рЃА рЃЦрЃЋрЃћрЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ, рЃЏрЃљрЃерЃўрЃю C-рЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ A рЃЎрЃЋрЃџрЃљрЃЋ рЃЦрЃЋрЃћрЃАрЃўрЃЏрЃарЃљрЃЋрЃџрЃћ рЃўрЃЦрЃюрЃћрЃЉрЃљ B-рЃўрЃА рЃЊрЃљрЃюрЃљрЃарЃЕрЃћрЃюрЃў рЃюрЃљрЃгрЃўрЃџрЃерЃў.

38. A Ріѓ B РЄњ A РѕЕ C Ріѓ B РѕЕ C
рЃЌрЃБ A рЃљрЃарЃўрЃА B-рЃА рЃЦрЃЋрЃћрЃ»рЃњрЃБрЃцрЃў, рЃЏрЃљрЃерЃўрЃю A-рЃАрЃљ рЃЊрЃљ C-рЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃљрЃАрЃћрЃЋрЃћ рЃўрЃЦрЃюрЃћрЃЉрЃљ B-рЃАрЃљ рЃЊрЃљ C-рЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЦрЃЋрЃћрЃ»рЃњрЃБрЃцрЃў.

39. A Ріѓ B РЄњ A Рѕф C Ріѓ B Рѕф C
рЃЌрЃБ A рЃерЃћрЃўрЃфрЃљрЃЋрЃА B-рЃерЃў (A Ріѓ B), рЃЏрЃљрЃерЃўрЃю A-рЃА рЃЊрЃљ C-рЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ рЃЦрЃЋрЃћрЃ»рЃњрЃБрЃцрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ B-рЃА рЃЊрЃљ C-рЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљрЃерЃў


40. C = A \ B РЄњ B РѕЕ C = РѕЁ
рЃЌрЃБ C рЃбрЃЮрЃџрЃўрЃљ A \ B, рЃЏрЃљрЃерЃўрЃю B-рЃА рЃЊрЃљ C-рЃА рЃерЃЮрЃарЃўрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА.

41. A Рѕф B = РѕЁ РЄћ A = РѕЁ & B = РѕЁ
рЃЌрЃБ A-рЃА рЃЊрЃљ B-рЃА рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ рЃљрЃарЃўрЃА рЃфрЃљрЃарЃўрЃћрЃџрЃў (РѕЁ), рЃЏрЃљрЃерЃўрЃю рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ A рЃЊрЃљ B рЃфрЃљрЃџ-рЃфрЃљрЃџрЃЎрЃћ рЃфрЃљрЃарЃўрЃћрЃџрЃўрЃљ.

42. A \ B = РѕЁ РЄћ A РѕЕ B = A
рЃЌрЃБ ­Юљ┤Рѕќ­Юљх=РѕЁ, рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ ­Юљ┤-рЃерЃў рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃўрЃАрЃћрЃЌрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃа рЃљрЃарЃўрЃА ­Юљх-рЃерЃў. 
рЃћрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃўрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЌрЃБ ­Юљ┤-рЃА рЃДрЃЋрЃћрЃџрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў ­Юљх-рЃерЃўрЃф рЃљрЃарЃўрЃА, рЃљрЃюрЃБ ­Юљ┤Ріє­Юљх. 
рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ, ­Юљ┤РѕЕ­Юљх=­Юљ┤. рЃљрЃАрЃћрЃЋрЃћ, рЃарЃћрЃфрЃўрЃърЃарЃЮрЃЎрЃБрЃџрЃљрЃЊ, рЃЌрЃБ ­Юљ┤РѕЕ­Юљх=­Юљ┤, ­Юљ┤-рЃА рЃДрЃЋрЃћрЃџрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃљрЃарЃўрЃА ­Юљх-рЃерЃў, рЃарЃљрЃф рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ ­Юљ┤Рѕќ­Юљх=РѕЁ.

43. A \ B = A РЄћ B \ A = B
рЃЌрЃБ ­Юљ┤Рѕќ­Юљх=­Юљ┤, рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ ­Юљх-рЃерЃў рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў ­Юљ┤-рЃАрЃЌрЃљрЃю (­Юљ┤РѕЕ­Юљх=РѕЁ). 
рЃљрЃЏрЃљрЃЋрЃћ рЃЊрЃарЃЮрЃА, ­ЮљхРѕќ­Юљ┤=­ЮљхрЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ ­Юљ┤-рЃерЃў рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў ­Юљх-рЃАрЃЌрЃљрЃю. рЃЮрЃарЃўрЃЋрЃћ рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃўрЃА рЃбрЃЮрЃџрЃцрЃљрЃАрЃўрЃљ.



рЃАрЃ«рЃЋрЃљ рЃЊрЃљрЃюрЃљрЃарЃЕрЃћрЃюрЃў рЃЋрЃћрЃа рЃњрЃљрЃЋрЃўрЃњрЃћ рЃЎрЃљрЃарЃњрЃљрЃЊ.
